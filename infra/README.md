# Infrastructure (Proxmox-first)

This folder contains infrastructure documentation and Ansible scaffolding used to reproduce the capstone lab environment.

Design goals:
- Proxmox-first (clinic-scale segmented lab)
- Linux hosts configured via Ansible (Level 2 automation)
- Windows endpoints documented (licensing and endpoint variability)
- Secrets provided externally (not committed to git)

See: `infra/ansible/README.md`
